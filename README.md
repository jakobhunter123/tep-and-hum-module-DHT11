# Vremenska postaja
Naprava meri temperaturo in vlaznost s pomocjo DHT11 senzor in ESP32 mikrokontrolerja

## Kosovnica


- ESP32
- DHT11 senzor
- OLED zaslon
- Kabelcki
- 9v Baterija
- 3D natisnjeno ohisje, ki je iz plastike
- 3x LED 
- Stikalo za vzig naprave
